#### T1

二分，转换为判定能否在前 $$k$$ 秒阻止疫情蔓延。

注意到，如果你在第 $$k$$ 秒一次性船访 $$k$$ 个国家能阻止，那么你前 $$k$$ 秒每秒选一个也能阻止。

所以就只需要先求出第 $$k$$ 秒时的感染情况，然后算一下要封多少个国家才能阻止蔓延。

显然我们会选择被感染的国家的相邻的未被感染的国家来封城。

时间复杂度 $$O(n\log n)$$

(数据足够水，暴力也可以过)

#### T3

最短路，但是有一种特殊的地铁边，`u v t0 t w` ，t0 秒发车，每 t 秒重新发，走 w 秒。

其实不影响。转移的时候只要 地铁不会破坏最短路的性质。 原 dist\[u] + w\[i] -> dist\[v] dist\[u] + w\[i] + (u后面第一个t0+t) -> dist\[v] 计算地铁的边权时考虑时间

```cpp
ll calc(ll t, int t0, int p, int cst)
{
    if (!p) return t + cst;
    t = max(t, 1LL * t0);
    t = t0 + (t - t0 + p - 1) / p * p;
    return t + cst;
}
```

直接跑 Dijkstra 就好了。
